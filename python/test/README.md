# PuMApy Tests

This folder contains the unit-tests of the PuMApy functions. 

# How to use
Tests are written using the unittest package and can be run singularly as e.g.: 

    python -m unittest test_workspace.TestWorkspace.test_WScreation
    
or class-wise as e.g.:

    python -m unittest test_workspace.TestWorkspace

or in bulk as:

    python -m unittest
